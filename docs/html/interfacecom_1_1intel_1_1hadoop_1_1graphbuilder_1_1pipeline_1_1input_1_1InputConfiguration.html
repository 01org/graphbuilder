Detailed Description

This interface encapsulates the methods used to configure an end-to-end map reduce chain so that it will work with the input format specified.

The first map job corresponds to stepping through the raw input and spitting out property graph elements. The input determines the first mapper.

The output determines the first reducer and any subsequent Map Reducer tasks in a chain.

The methods in this interface are used by the full Map Reducer chain to properly configure the first Map Reducer job to work with the input mapper.
