Detailed Description

VertexList UDF intakes property graph elements and spits out vertex list elements which are tuples of (vertex id, vertex label). For example, (Employee001, OWL.People) If the constructor parameter is set to "TRUE", "true" or "1", the vertex list element also contains edge properties returned as key value pair, e.g. (Employee002, OWL.People, name:Alice, age:30)
