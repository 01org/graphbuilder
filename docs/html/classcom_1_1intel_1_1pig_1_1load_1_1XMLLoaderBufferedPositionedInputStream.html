Detailed Description

A XMLLoaderBufferedPositionedInputStream is the package class and is the decorator over the BufferedPositionedInputStream which in turn decorate BufferedInputStream. It contains BufferedPositionedInputStream input stream, which it uses as its basic source of data, possibly reading or providing additional functionality. The class XMLLoaderBufferedPositionedInputStream itself simply overrides the necessary method for reading i.e read getPosition with versions that pass all requests to the contained input stream or do some special processing. Subclasses of XMLLoaderBufferedPositionedInputStream may further override some of these methods and may also provide additional methods and fields. It also provides additional method collectTag<collect> which will give the byte array between the tag which is a xml record. i.e <tag> .*</tag> will be returned

Note:
we can't use the standard SAX or STAX parser as for a big xml the intermittent hadoop block may not be the valid xml and hence those parser may create pb.
