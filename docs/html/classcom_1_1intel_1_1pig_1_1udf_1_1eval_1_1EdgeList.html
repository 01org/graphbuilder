Detailed Description

EdgeList UDF intakes property graph elements and spits out edge list elements which are tuples of (edge source vertex id, edge target vertex id, label), for example, (Employee001, Employee002, worksWith) If the constructor parameter is set to "TRUE", "true" or "1", the edge list element also contains edge properties returned as key value pair, e.g. (Employee001, Employee002, worksWith, workedFor:11yrs)
